## What I've Shipped

### [White Glove Wireless](https://whitegwireless.com) — AI-Powered SDR Platform

A full production sales platform I designed, built, and deployed end-to-end
for an AT&T authorized dealer — zero engineers, just me.

- **AI cold calling** via Bland.ai — bilingual EN/ES, auto-detects language mid-call
- **Outcome detection** — webhooks + Claude API parse transcripts in any language,
  auto-book appointments, mirror to Google Calendar via OAuth
- **Lead sourcing** — Google Places API, territory assignment, rep management
- **SMS automation** — Twilio with TCPA/DNC compliance gate + per-state disclosure
- **Compliance tooling** — call recording disclosures, soft-delete logs,
  per-state retention settings (admin-entered)
- **Multi-tenant fork** — white-label SaaS architecture, metered billing

**Stack:** Node · Express · React · Supabase · Railway · Vercel ·
Bland.ai · Twilio · Claude API · Google OAuth
